### About Me

I'm **Cristian Dolință** — systems administrator and DevOps engineer at **QIWI Moldova**, managing a three-datacenter infrastructure across Kubernetes, VMware, MikroTik, and custom SIEM pipelines.

Outside work I run **CDL Production**, an independent mobile studio shipping Flutter apps for iOS & Android — faith-based tools, productivity apps, and arcade games.
